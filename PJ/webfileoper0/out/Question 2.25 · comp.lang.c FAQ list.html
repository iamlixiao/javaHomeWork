Q:I came across some structure declarations with colons and numbers next to certainfields,like this:

struct record {	char *name;	int refcount : 4;	unsigned dirty : 1;};
What gives?

A:Those are bit-fields;the number gives the exact size of the field,in bits.(See any complete book on C forthe details.)Bit-fields can be used to save spacein structures having several binary flags or othersmall fields,and they can also be usedin an attempt to conform to externally-imposed storage layouts.(Their success at the latter task is mitigatedby the fact that bit-fields are assigned left-to-right on some machines and right-to-left on others).

Note that the colon notationfor specifying the size of a field in bitsis only valid in structures(and in unions);you cannot use this mechanismto specify the size of arbitraryvariables.(See questions 1.2 and 1.3.)
