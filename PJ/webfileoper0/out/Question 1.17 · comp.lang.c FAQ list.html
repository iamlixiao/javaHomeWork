Q:What does

typedef int (*funcptr)();
mean?

A:It defines a typedef, funcptr, for a pointer to afunction(taking unspecified arguments)returning an int.It can be used todeclareone or morepointers to functions:

	funcptr pf1, pf2;
which is equivalent to the more verbose,and perhaps harder to understand
	int (*pf1)(), (*pf2)();
See also questions1.21,4.12,and15.11.
