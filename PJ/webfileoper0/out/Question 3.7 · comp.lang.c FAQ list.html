Q:Why did

printf("%d %d", f1(), f2());
call f2 first?I thought the comma operator guaranteed left-to-right evaluation.

A:The comma operator does guarantee left-to-right evaluation,but the commas separating the arguments in a function callare not comma operators.[footnote]The order of evaluation of the arguments to a function callis unspecified.(See question 11.33.)
