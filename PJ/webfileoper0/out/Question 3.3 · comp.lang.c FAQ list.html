Q:I've experimented withthe code

int i = 3;i = i++;
on several compilers.Some gave i thevalue 3,andsome gave 4.Which compiler is correct?

A:There is no correct answer;theexpression is undefined.See questions3.1,3.8,3.9,and11.33.(Also, note that neither i++ nor ++iis the same as i+1.If you want to increment i,usei=i+1,i+=1,i++,or++i,not some combination.See also question 3.12b.)
