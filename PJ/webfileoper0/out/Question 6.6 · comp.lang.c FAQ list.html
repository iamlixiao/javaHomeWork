Q:If you can't assign to arrays,thenhow can

	int f(char str[])	{		if(str[0] == '\0')			str = "none";		...	}
work?

A:In this code,str is a function parameter,so its declaration is rewritten by the compileras described in question 6.4.In other words, str is a pointer(of type char *),and it is legal toassign toit.
